FixedValueList Object - Expressions

Get the current fixed value
This expression gets the currently iterated fixed value. This will be different for every object.

Do not mix up LastFixedValue("FixedValueList") with Fixed("FixedValueList") - they are completely different. You can use the following condition to check that the current object is the type you want to do the actions to:

General object>Compare two general values:
LastFixedValue("FixedValueList") Is Equal To Fixed("Object")
